## Recharge Determining Factors (Raster Data)

This directory contains raster datasets used as input variables for pond-based groundwater recharge modelling in the Ramganga Basin (RGB), India.

All datasets are preprocessed, spatially aligned, and resampled to a common resolution.

### Folder Structure

- **climate/** → Precipitation, Temperature, ET, Aridity Index  
- **hydrology/** → Groundwater Level, Infiltration, Surface Runoff, TWI  
- **soil/** → Soil Moisture, Permeability, Soil Texture, Organic Content  
- **land_surface/** → NDVI, Curve Number, Slope  

### Format
GeoTIFF (.tif)

### Note
Due to file size limitations, only sample datasets are provided here. Full datasets are available upon request or via external repository.

These layers were used as predictors in statistical and machine learning models for recharge estimation.
