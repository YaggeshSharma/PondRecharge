## Recharge Determining Factors (Raster Attributes)

This directory contains attribute tables (.dbf format) corresponding to raster datasets used as determining factors for pond-based groundwater recharge modelling in the Ramganga Basin (RGB), India.

Due to file size limitations, the full raster datasets (.tif) are not included in this repository. Instead, the associated attribute tables are provided to facilitate inspection of variable distributions and values.

### Available Layers
The following factors are included:

 
- Curve Number  
- Groundwater Level (GWL)   
- NDVI  
- Permeability  
- Rainfall 
- Soil Organic Matter 
- Soil Moisture  
- Temperature  
- Turbidity Index 
- Topographic Wetness Index (TWI)  

### Data Format
- Attribute tables: `.dbf`  
- Raster format (not included): `.tif`  

### Notes
- All raster layers were preprocessed, resampled, and aligned to a common spatial resolution prior to modelling.
- These variables were used as input features in statistical and machine learning models for recharge estimation.
- Full raster datasets are available upon request or via an external data repository.
