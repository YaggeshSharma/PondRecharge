## Google Earth Engine (GEE) Scripts for Pond Inventory

This folder contains Google Earth Engine (GEE) scripts used to generate the pond inventory for the Ramganga Basin (RGB), India.

### Purpose
The scripts were developed to identify and extract pond locations using remote sensing techniques. Water indices such as NDPI, NDWI, and MNDWI were applied to satellite imagery to detect surface water bodies and map unregistered ponds.

### Data Source
- Landsat 9 TOA imagery  
- Study area boundary (Ramganga Basin shapefile)

### Methodology
- Image filtering based on date and cloud cover  
- Calculation of water indices (NDPI, NDWI, MNDWI)  
- Threshold-based extraction of water pixels  
- Clipping to basin boundary  
- Export of raster outputs for further processing  

### Output
- Water index rasters (GeoTIFF)  
- Extracted pond pixels used for pond inventory development  

### Notes
- Threshold values (e.g., NDWI > 0) may be adjusted based on local conditions  
- Users should update asset paths and export settings before running the scripts  
- Final pond inventory was generated by combining remote sensing outputs with government datasets and field observations
